PROCEDURE TO RUN THE APPLICATION
--------------------------------

1. Import the LinkedinR.sql file into your local PHP MY ADMIN server.

2. Make sure there are no errors in the database. If there are any, please correct them accordingly.

3. Now, go to Google Chrome and type http://localhost/projectGroup8/aboutus.php

4. click on Register and it should take you to the registration page. 

5. Register as an EMPLOYEE OR EMPLOYER. Proceed with the flow of the application

6. Next time, just click on Login from the aboutus.php page and connect to users and employees.
